About Us

ComicPub a.k.a Comic Publish is a fully deployable web app template written in DJango by Xonshiz. Specially made for Manga Scanlation groups and comic hosters. You don't have to store all the chapter images on your web server and serve them to your viewers anymore. You can utilize the power of Imgur albums and give your users a nice experience.

This app neither violates Imgur's TOS, nor intends to do so in any possible way. At the time of writing this app, according to Imgur, direct hotlinking of images is NOT allowed. But, embeding an album is "okay" (Since they can generate revenue from it), cool enough.

Since direct hotlinking is not allowed, we cannot use the image's direct link to host the Cover Images of the Comics, so that you'll have to store on your web server (which, I think is pretty much OKAY).

Since this is a Django app, you cannot host it on normal web hosts. You need Django Hosting. There are two options in such case :

Paid Hosting For Django :
- DigitalOcean
- Heroku
- Google Cloud Platflorm
- Amazon AWS


Free Hosting For Django :
- HelioHost
- PythonAnywhere
- Heroku (Has Limits)
- Amazon AWS (Free Tier, Has Some Limits)
